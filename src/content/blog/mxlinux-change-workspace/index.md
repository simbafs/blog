# Mxlinux Change Workspace

## 切換工作區

今天想要在 mxlinux 中快速的切換工作區
我看到可以用滑鼠滾輪切換
可是我想要的是鍵盤快捷鍵
上網查了一下
發現可以透過 `Ctrl`+`Alt`+`up/down/home/end` 來切換
